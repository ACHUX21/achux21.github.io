# Sanity Check

## Challenge Context

Sanity Check was the event warm-up challenge for 1337 Local CTF. The purpose was not technical exploitation; it verified that players had joined the event communication channel and knew the flag format.

## Reconnaissance

The workspace notes record this as solved from Discord:

## Core Idea

Sanity checks usually place the flag in the rules page, Discord announcement, or event description. The correct method is to read the organizers' communication channel instead of trying to attack infrastructure.
