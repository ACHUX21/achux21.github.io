## Stuff I've Built

**[packguard](https://github.com/ACHUX21/packguard)** — Offline supply chain security scanner. Runs local AI (Ollama) against npm and PyPI packages to catch malicious code, typosquatting, and suspicious patterns. No cloud, no telemetry, just static analysis and heuristics.

**[checker-CVE-2024-6387](https://github.com/ACHUX21/checker-CVE-2024-6387)** — Detection script for the regreSSHion vulnerability (CVE-2024-6387). Targeted OpenSSH servers, checks for the race condition without triggering it.

**[Hyper-dotfile](https://github.com/ACHUX21/Hyper-dotfile)** — Terminal setup I actually use. Hyper Terminal + OhMyZsh + Powerlevel10k, tuned for dark purple aesthetics. 9 stars, people seem to like it.

**[BlackArch-Repo](https://github.com/ACHUX21/BlackArch-Repo)** — Scripts and repo management for BlackArch Linux. Collection of tools I've put together during pentesting work.

**[Low_level_programming](https://github.com/ACHUX21/Low_level_programming)** — Systems programming in C. Memory management, pointers, data structures — the stuff that teaches you how computers actually work.

**[simple_shell](https://github.com/ACHUX21/simple_shell)** — Custom Unix shell written from scratch. Parsing, process management, execution flow. Built it to understand what happens when you type a command.
